ℹ️ About This Build

Built by VK4AHA for the Hack Club Stardance program. Live data sources: aprs.fi (station tracking, key required), POTA/SOTA public spot APIs, wspr.live, and dxsummit.fi. Satellite passes use live TLEs from Celestrak with SGP4 propagation via satellite.js, computed client-side — no backend, no tracking, no accounts.
