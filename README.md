Aus Ham Radio Connect

This program is designed for HAM Radio Operations, Specifically carried out in australia, designed for the Hackclub Stardance Project,
It allows HAM Radio Operators to coordinate with a Ham Clock Inspired system
It allows for integrations like
   - APRS.fi Integration  (requires free API token)
   - VK/ZL POTA Spotting Tracking
   -  A integrated suite of tools in regards to HAM Radio Calculators (SWR, Antennas, etc)
   -  A band reference system
   -  Phonetics and Q-Codes Reference
   -  Integrated offline beacon tracker (based of intervals)
   -  UTC Clock
   -  Maidenhead Calculator
   -  Satelite Pass Calculator
   -  Live DX Cluster Integration
   -  WIA and Repeater Maps (offline, data cached in your browser!)

You can check it out at https://sammakesstuff.github.io/Aus-Ham-Radio-Connect/


How to run it locally

You can clone the repo, and run it using integrated html (some of the integrations may not work offline)
