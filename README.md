So, yeah here it is, Aus Ham Radio Connect, 
This was made by me and claude for a hackclub stardance project, to get myself introduced into using systems like this, I hope that this helps someone else with ham radio, 

Some of the integrations include


- APRS tracking (requires free API key from aprs.fi)
- UTC, Sunrise & Sunset Tracking
- A integrated suite of tools in regards to repeater maps, club maps sourced from the wireless institute of australia and ACMA

A lot of this project is vibecoded with claude... but i hope it helps someone :3 
You can clone this repo and run dashboard.html on a local webserver to use it. or via github pages
